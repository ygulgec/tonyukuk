Özellikler Başlangıç Dil Referansı Kütüphane Matris İstatistik Playground

Tonyukuk

Tamamen Türkçe sözdizimi ile programlama. Derlenen, hızlı ve modern bir programlama dili.

Hemen Başla Playground
örnek.tr
# Fibonacci hesapla işlev fibonacci(n: tam) -> tam eğer n <= 1 ise döndür n son döndür fibonacci(n - 1) + fibonacci(n - 2) son döngü i = 0, 10 ise yazdır(fibonacci(i)) son

Özellikler

Türkçe Sözdizimi

Tüm anahtar sözcükler, tip isimleri ve yerleşik fonksiyonlar Türkçe. Programlamayı kendi dilinizde öğretin ve yazın.

Derlenen Dil

x86_64 assembly'ye derlenir. GCC ile bağlama yapılır. Hızlı ve verimli çalışma zamanı.

Zengin Tip Sistemi

Tamsayı, ondalık, metin, mantık, dizi ve sözlük tipleri. Sınıflar, kalıtım ve arayüz desteği.

Modern Özellikler

String interpolasyon, pipe operatörü, desen eşleme (eşle/durum), lambda ifadeleri ve daha fazlası.

Modül Sistemi

Matematik, metin, sistem, dizi, zaman, JSON, ağ ve daha fazla yerleşik modül. Dosya modülleri ile kod organizasyonu.

Geliştirici Araçları

Kod biçimleyici, denetleyici (linter), paket yöneticisi, VSCode eklentisi ve REPL modu.

Hızlı Başlangıç

1

Kaynak kodu indirin ve derleyin

git clone https://github.com/tonyukuk/derleyici.git cd derleyici make
2

İlk programınızı yazın (merhaba.tr)

yazdır("Merhaba Dünya!")
3

Derleyin ve çalıştırın

./tonyukuk-derle merhaba.tr -o merhaba ./merhaba

Dil Referansı

Değişkenler ve Tipler
tam sayı = 42 ondalık pi = 3.14 metin ad = "Tonyukuk" mantık aktif = doğru dizi liste = [1, 2, 3]

İşlevler
işlev topla(a: tam, b: tam) -> tam döndür a + b son

Kontrol Akışı
eğer sayı > 0 ise yazdır("Pozitif") yoksa yazdır("Negatif") son

Sınıflar
sınıf Hayvan tam yaş işlev bilgi() -> tam döndür bu.yaş son son

Döngüler
# For döngüsü döngü i = 1, 10 ise yazdır(i) son # While döngüsü tam sayaç = 0 iken sayaç < 5 ise sayaç += 1 son # For-each dizi d = [10, 20, 30] her eleman için d ise yazdır(eleman) son

Hata Yönetimi
dene fırlat 42 yakala hata yazdır(hata) son

Eşle / Durum (Pattern Matching)
tam puan = 85 eşle puan / 10 ise durum 9: yazdır("Pekiyi") durum 8: yazdır("İyi") durum 7: yazdır("Orta") varsayılan: yazdır("Diğer") son

sayisal_analiz_örnek.tr

# Polinom: katsayılar [a, b, c] → ax² + bx + c
dizi k = [1, 0, -4]           # x² - 4
yazdır(polinom_hesapla(k, 3))  # 5  (3² - 4)

# Türev: x² - 4 → 2x → [2, 0]
dizi td = polinom_turev(k)
yazdır(td[0])                   # 2
yazdır(td[1])                   # 0

# İntegral: 3x² + 2x + 1 → x³ + x² + x + C
dizi k2 = [3, 2, 1]
dizi ig = polinom_integral(k2)  # [1, 1, 1, 0]

# Sayısal integral (yamuk kuralı)
dizi xi = [0, 1, 2, 3, 4]
dizi yi = [0, 1, 2, 3, 4]
yazdır(sayisal_integral(xi, yi)) # 8 (y=x üzerinde [0,4])

# Lineer regresyon
dizi x = [1, 2, 3, 4, 5]
dizi y = [2, 4, 6, 8, 10]
dizi lr = lineer_regresyon(x, y)
yazdır(lr[0])                   # 2 (eğim)
yazdır(lr[1])                   # 0 (kesişim) → y = 2x

10. Komut Satırı Kullanımı

Tonyukuk derleyicisi komut satırı seçenekleri.

Kullanım
tonyukuk-derle [seçenekler] <dosya.tr>

Seçenekler:
  -o <dosya>       Çıktı dosya adı
  -S               Sadece assembly üret (.s dosyası)
  -c               Sadece nesne dosyası üret (.o dosyası)
  --ast            Soyut sözdizim ağacını (AST) göster
  --sembol         Sembol tablosunu yazdır
  --optimize       Optimizasyonları etkinleştir
  --hata-ayıkla    Hata ayıklama bilgisi ekle
  --sürüm          Derleyici sürümünü göster
  --yardım         Yardım mesajını göster

Örnekler
# Temel derleme
tonyukuk-derle merhaba.tr -o merhaba

# Assembly çıktısı al
tonyukuk-derle merhaba.tr -S

# Optimizasyon ile derle
tonyukuk-derle program.tr -o program --optimize

# AST çıktısını incele
tonyukuk-derle program.tr --ast

# Hata ayıklama modunda derle
tonyukuk-derle program.tr -o program --hata-ayıkla
